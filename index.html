Руководство пользователя
    1. Нажать на кнопку скачать
    2. Запустить скачанный .exe файл
Правила тренажера
    1. Если правильно ввести строку, то новая строка уменьшиться на 2
    2. Уровень заканчивается, когда новая строка становится нулевой
    3. Каждый уровень сложнее предыдущего. Каждый неправильно введенный символ увеличивается в 2*(номер раунда) раз
    4. В случае частично верного написания строки каждое правильно введенное слово заменяется
    5. Если из-за ошибок строка станет слишком длинной, то уровень придется начать с начала
    6. Введите '13', чтобы закончить тренировку
